# PySpark Trump Tweets Analysis

## Overview
This project analyzes large-scale text data using PySpark to extract communication patterns and statistics from political tweets.

## Dataset
Dataset of approximately 58,000 tweets from Donald Trump between 2009 and 2016.

## Methodology
- Distributed data processing with RDDs
- Text preprocessing
- Frequency and pattern analysis

## Tools & Technologies
- PySpark
- Python

## Key Takeaways
The project showcases big data processing skills and scalable analytics using Spark.
